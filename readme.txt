Compresor de Huffman.

    Compresor implementado en C  basado en el algoritmo de codificacion de Huffman.

Construcción:

    $ make program

Compresión:

    [$ ./huff C file  ]
    
        Devuelve: (f.txt.hf) y (f.txt.tree) 

Descompresión: (Necesario que existan los archivos (f.txt.hf) y (f.txt.tree)

    [$ ./huff D  ]
